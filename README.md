Default.rb Template Usage
=========================
* Run `gem install rails --no-ri --no-rdoc`
* Run `rails new APP_NAME --skip-prototype --skip-test-unit --template=/path/to/default.rb`
* Run `rake db:create`
* Run `rake db:migrate`
* Run `rails s`
* Browse to `http://localhost:3000/`
